# Company Core Knowledge

This directory contains the foundational knowledge every agent needs to operate —
before reading any playbook, skill, or task.

**Live source of truth:** Lark Wiki → Company Core Knowledge
**This copy:** reproducible backup for new instance setup

---

## How to use this

1. **Agents** — read these files at the start of any session involving client-facing work,
   content creation, or sales. The PIC file is the most critical for escalation decisions.

2. **Humans** — fill in `{{PLACEHOLDER}}` values when setting up a new instance.
   Keep the Lark Wiki as the live version; sync changes here periodically.
